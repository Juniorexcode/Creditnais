# Creditnais

Creditnais is a modern, premium financial landing page designed to showcase an innovative prize-linked debit card. Built with cutting-edge web technologies, it features an immersive, dynamically animated user experience powered by React, Tailwind CSS, and GSAP.

## ✨ Features

- **Immersive GSAP Animations:** A custom scroll-linked animation system that gracefully transitions elements as you scroll through the page.
- **Glassmorphism Design:** Premium UI elements with beautiful glass effects, soft shadows, and deep navy/blue gradients.
- **Dynamic Backgrounds:** Mathematical precision in background elements that smoothly glide and adapt across different viewport heights.
- **Responsive Layout:** Beautifully structured to provide a rich experience across both desktop and mobile devices.

## 🚀 Getting Started

To run this project locally:

1. **Install Dependencies:**
   ```bash
   npm install
   ```
2. **Start the Development Server:**
   ```bash
   npm run dev
   ```
3. Open `http://localhost:5173` to view it in the browser.
